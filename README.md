# Projeto de Testes Automatizados - Mercury Tours

Este repositório contém um projeto de testes automatizados para o site Mercury Tours. 
Os testes foram desenvolvidos utilizando as seguintes tecnologias e ferramentas:

- **Eclipse**: Ambiente de desenvolvimento integrado (IDE)
- **Maven**: Gerenciamento de dependências e automação de build
- **Selenium**: Framework para automação de navegadores
- **JUnit**: Framework para testes unitários
- **Page Object Pattern**: Padrão de design para organização de código de testes

## Funcionalidades

- **Cadastro de Usuário**: Testes automatizados que verificam o processo de cadastro de um novo usuário no site Mercury Tours.

## Estrutura do Projeto

- `src/main/java`: Contém as classes principais do projeto.
- `src/test/java`: Contém as classes de teste e os Page Objects.
- `pom.xml`: Arquivo de configuração do Maven com as dependências do projeto.
